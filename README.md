# Overview
This project is making for final quiz native mobile subject

# Install
Make sure you have android SDK
1. clone project
```
git clone https://github.com/chaiyut-kun/flutter_final_quiz.git
```
```
cd flutter_final_quiz
```
2. install dependecies
```
flutter pub get
```

3. run with emulators
- run your emulators
```
flutter run
```
